# 🚀 SaaSFlow - Modern SaaS Landing Page

A beautiful, responsive landing page template for SaaS products built with Tailwind CSS and vanilla JavaScript.

![SaaSFlow Preview](assets/screenshots/preview.png)

## ✨ Features

- 🎨 Modern gradient design with glassmorphism
- 📱 Fully responsive (mobile, tablet, desktop)
- ♿ WCAG 2.1 AA accessibility compliant
- ⚡ Lightweight (no frameworks except Tailwind CDN)
- 🎭 Smooth animations and transitions
- 🌙 Dark theme optimized
- 🔧 Easy to customize

## 🎯 Sections

- Hero with animated dashboard preview
- Statistics counter
- Feature cards with hover effects
- How It Works timeline
- Pricing with monthly/yearly toggle
- Customer testimonials
- FAQ accordion
- Newsletter signup
- Responsive footer
